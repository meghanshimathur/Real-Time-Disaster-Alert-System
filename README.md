# 🌍 Real-Time Disaster Alert System using Agentic AI & Machine Learning

## 📌 Project Overview
This project implements a real-time disaster alert system that collects data from multiple live sources, identifies disaster events, classifies their type and severity, and generates safety advisories using Generative AI.

The system is designed using an **agentic AI architecture**, where each component (agent) performs a specific task such as data ingestion, classification, severity analysis, and advisory generation.

---

## 🎯 Objectives
- To aggregate disaster-related data from multiple reliable sources
- To apply machine learning for disaster classification
- To estimate severity and priority of disaster events
- To generate human-readable safety advisories using Generative AI
- To visualize alerts through a simple web-based interface

---

## 🧠 Technologies Used
- **Python**
- **Flask** (Frontend)
- **Machine Learning (NLP)** – Transformer-based classification
- **Generative AI** – Google Gemini
- **Pandas** – Data handling
- **Matplotlib** – Analytics & visualization
- **REST APIs** – USGS, NewsAPI, NewsData
